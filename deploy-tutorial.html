第7步：（可选）neecai.net 自动跳转 www.neecai.net

7

让不带 www 的地址也能访问

  1. Cloudflare 后台 → 左侧 网站 → 点 neecai.net
  2. 左侧菜单 → 规则 → 重定向规则
  3. 点 创建规则
  4. 设置：
    • 名称：www 跳转
    • 匹配条件：主机名 = neecai.net
    • 类型：选 动态
    • 值填：concat("https://www.neecai.net", http.request.uri.path)
    • 状态码：301
  5. 点 部署

这样别人输入 neecai.net 就会自动跳转到 www.neecai.net

第8步：配置询盘表单（收到客户邮件）

8

注册 Formspree

  1. 打开 👉 formspree.io → 注册账号
  2. 点 New Form（新建表单）→ 名字填 neecai-inquiry
  3. 设置接收邮箱（填你自己的邮箱）
  4. 复制表单 ID（类似 mrgvwpql 的一串字符）

把表单 ID 填到网站代码里

  1. 打开 👉 github.com/sophy00189/neecai-website
  2. 点 index.html 文件
  3. 点右上角 ✏️ 编辑按钮
  4. 按 Ctrl+F 搜索 YOUR_FORM_ID
  5. 把 YOUR_FORM_ID 替换成你从 Formspree 复制的表单 ID
  6. 点 Commit changes（提交修改）
  7. 等1-2分钟自动部署

❓ 常见问题

Q: 部署失败，显示红色错误？

A: 99%是第4步构建设置填错了。回去检查：框架预设= None、构建命令=留空、输出目录= /

Q: .pages.dev 能打开，但 www.neecai.net 打不开？

A: 域名还没绑定，回到第6步操作。或者 DNS 还没生效，等几分钟。

Q: 页面打开是空白的？

A: 构建输出目录填错了。应该是 /，不是 ./dist 或 ./build。

Q: 以后怎么改网站内容？

A: 去 GitHub 仓库 → 点文件 → 点 ✏️ 编辑 → 改完 Commit → 1-2分钟自动上线。

Q: 表单提交后没收到邮件？

A: 检查是否把 YOUR_FORM_ID 替换成了 Formspree 的实际 ID。检查垃圾邮件箱。
