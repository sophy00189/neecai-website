🔧 添加 DNS 记录教程

让 www.neecai.net 指向你的网站

⚠️ 当前状态

网站已成功部署在 https://neecai-website.pages.dev，但 www.neecai.net 还无法访问，因为缺少一条 DNS 记录。只需 2 分钟即可搞定！

1 登录 Cloudflare 并进入 DNS 设置

1. 打开 Cloudflare 控制台：https://dash.cloudflare.com

2. 点击你的网站 neecai.net

3. 在左侧菜单中点击 DNS → Records

dash.cloudflare.com → neecai.net → DNS → Records

2 点击"添加记录"按钮

点击页面右上角的 + 添加记录 按钮，会弹出一个表单

3 填写 CNAME 记录

按照下面的信息填写表单：

💡 重要提示：代理状态一定要选择 "已代理 (Proxied)"，图标是 🟠 橙色云朵，不要选成"仅 DNS (DNS only)" 灰色云朵。Proxied 模式才能使用 Cloudflare 的 CDN 加速和 SSL 证书。
⬇️

4 点击"保存"

点击 保存 按钮完成添加。

添加后，DNS 记录列表应该变成这样：

dash.cloudflare.com → neecai.net → DNS → Records

5 验证网站

保存后等待 1-5 分钟，然后打开浏览器访问：

https://www.neecai.net

如果能看到你的舞台特效网站，就说明配置成功了！🎉

📝 后续待办事项

1. Formspree 表单配置：当前联系表单的 YOUR_FORM_ID 需要替换为真实的 Formspree ID，否则表单提交不生效

2. 根域名重定向：建议将 neecai.net 自动跳转到 www.neecai.net（可在 Cloudflare 的 Page Rules 中设置）

3. 安全提醒：建议删除之前在聊天中分享的 GitHub Token 和 Cloudflare API Token
